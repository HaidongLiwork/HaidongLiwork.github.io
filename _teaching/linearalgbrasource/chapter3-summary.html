📝 第三章 · 本章总结

一、初等变换与标准形
  • 三种行初等变换：交换两行 (\(r_i \leftrightarrow r_j\))、某行乘非零常数 (\(r_i \times k\))、某行倍加至另一行 (\(r_i + k r_j\))。列变换同理。
  • 行阶梯形矩阵：非零行在零行之上，且每行首元位于上一行首元的右侧。
  • 行最简形矩阵：阶梯形 + 首元为1 + 首元所在列其余元素全为0。唯一。
  • 标准形：\[ F_r = \begin{pmatrix} E_r & 0 \\ 0 & 0 \end{pmatrix} \]，由初等变换（行+列）得到，唯一。
二、矩阵的秩
  • 定义一：标准形 \(F_r\) 中的 \(r\)。
  • 定义二：最高阶非零子式的阶数。两种定义等价。
  • 核心性质：
    • 初等变换不改变秩：\(A \sim B \Rightarrow R(A)=R(B)\)。
    • \(R(A^T)=R(A)\)，\(0 \le R(A) \le \min\{m,n\}\)。
    • 分块：\(R(A) \le R(A,B)\)，\(R(A+B) \le R(A)+R(B)\)。
    • 乘积：\(R(AB) \le \min\{R(A),R(B)\}\)。
    • 若 \(AB=0\)（\(A\) 为 \(m\times n\)），则 \(R(A)+R(B) \le n\)。
  • 方阵可逆 \(\iff R(A)=n\)。
三、线性方程组的解
  • 对 \(Ax=b\)，增广矩阵为 \((A,b)\)，判别如下：
    • \(R(A) < R(A,b)\)：无解。
    • \(R(A) = R(A,b) = n\)：有唯一解。
    • \(R(A) = R(A,b) < n\)：有无穷多解（自由未知量个数为 \(n-r\)）。
  • 齐次特例：\(Ax=0\) 有非零解 \(\iff R(A) < n\)。
  • 矩阵方程：\(AX=B\) 有解 \(\iff R(A)=R(A,B)\)。
四、本章核心思想

完整呈现了数学 “从具体到抽象，再回到具体” 的研究范式：解方程（具体）→ 初等变换与秩（抽象）→ 解的判别（具体，但眼界已大不同）。秩是矩阵在初等变换下的 不变量，也是本章的理论核心。
