🐍 Python 实现 —— 公安情报矩阵运算

常用于特征加权、关联强度计算、批量相似度比对等实战任务。以下提供两种实现方式。

纯 Python (三重循环)
def matmul_pure(A, B):
    m, n = len(A), len(A[0])
    p = len(B[0])
    C = [[0] * p for _ in range(m)]
    for i in range(m):
        for j in range(p):
            s = 0
            for k in range(n):
                s += A[i][k] * B[k][j]
            C[i][j] = s
    return C

# 示例：情报特征矩阵 × 线索权重矩阵
A = [[1, 2, 3],
     [4, 5, 6]]
B = [[7, 8],
     [9, 10],
     [11, 12]]
C = matmul_pure(A, B)
print(C)  # [[58, 64], [139, 154]]
NumPy (高效运算)
import numpy as np

A = np.array([[1, 2, 3],
              [4, 5, 6]])
B = np.array([[7, 8],
              [9, 10],
              [11, 12]])

C = np.dot(A, B)   # 或 A @ B
print(C)
# 输出:
# [[ 58  64]
#  [139 154]]
