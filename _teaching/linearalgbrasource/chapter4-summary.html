📝 第四章 · 本章总结

一、向量及其基本概念
  • \(n\) 维向量：\(n\) 个有次序的数 \(a_1,a_2,\dots,a_n\)，可写成行向量或列向量。本讲义默认采用列向量。
  • 向量空间：\(\mathbb{R}^n = \{ (x_1,\dots,x_n)^T \mid x_i \in \mathbb{R} \}\)。
  • 向量组：若干个同维数向量组成的集合。有限向量组与矩阵一一对应。
二、线性组合与线性表示
  • 线性组合：\( k_1\boldsymbol{a}_1 + k_2\boldsymbol{a}_2 + \cdots + k_m\boldsymbol{a}_m \)。
  • 线性表示：\(\boldsymbol{b} = \lambda_1\boldsymbol{a}_1 + \cdots + \lambda_m\boldsymbol{a}_m\)。
  • 判别定理：\(\boldsymbol{b}\) 能由向量组 \(A\) 表示 \(\iff R(A) = R(A,\boldsymbol{b})\)。
  • 向量组等价：两个向量组能相互线性表示 \(\iff R(A)=R(B)=R(A,B)\)。
三、线性相关与线性无关
  • 定义：存在不全为零的 \(k_1,\dots,k_m\) 使 \(\sum k_i\boldsymbol{a}_i = \boldsymbol{0}\)，则线性相关；否则线性无关。
  • 矩阵判别：\(A=(\boldsymbol{a}_1,\dots,\boldsymbol{a}_m)\)，则
    • 线性相关 \(\iff R(A) < m\)；
    • 线性无关 \(\iff R(A) = m\)。
  • 几何意义：两个向量相关 ⇔ 共线；三个向量相关 ⇔ 共面。
四、重要定理
  • 若一组向量线性相关，则添加任意向量后仍线性相关（“相关则加相关”）。
  • \(m\) 个 \(n\) 维向量，当 \(m > n\) 时一定线性相关（“高个子”必相关）。
  • 若 \(\boldsymbol{a}_1,\dots,\boldsymbol{a}_m\) 线性无关，而添加 \(\boldsymbol{b}\) 后线性相关，则 \(\boldsymbol{b}\) 可由它们 唯一 线性表示。
五、三种语言的统一
矩阵  ⟷  向量组  ⟷  线性方程组

本章的核心思想是建立这三种语言之间的 等价翻译。同一个线性代数问题，可以从矩阵、向量组、方程组三个角度切入，选择最便利的工具解决。
