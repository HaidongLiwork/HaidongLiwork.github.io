习题 1   行列式的唯一性

第一小问

设 \(A=(\alpha_1,\dots,\alpha_n)\)，且第 \(k\) 列为零向量，即 \(\alpha_k=0\)。

由列线性，取 \(\lambda=0\)，有

\[ f(\alpha_1,\dots,0,\dots,\alpha_n) = f(\alpha_1,\dots,0\cdot \alpha_k,\dots,\alpha_n) = 0\cdot f(\alpha_1,\dots,\alpha_k,\dots,\alpha_n) =0. \]

所以若 \(A\) 有一列全为 \(0\)，则 \(f(A)=0\)。

第二小问

我们要证明：对任意 \(n\) 阶实矩阵 \(A\)，都有

\[ f(A)=\det(A). \]

先证明 \(f\) 在初等列变换下的变化规律与行列式完全相同。

1. 交换两列
设交换第 \(i\) 列和第 \(j\) 列。因为两列相同则 \(f=0\)，所以

\[ 0=f(\dots,\alpha_i+\alpha_j,\dots,\alpha_i+\alpha_j,\dots). \]

由列线性展开：

\[ 0=f(\dots,\alpha_i,\dots,\alpha_i,\dots) +f(\dots,\alpha_i,\dots,\alpha_j,\dots) +f(\dots,\alpha_j,\dots,\alpha_i,\dots) +f(\dots,\alpha_j,\dots,\alpha_j,\dots). \]

其中第一项和第四项都为 \(0\)，因此

\[ f(\dots,\alpha_i,\dots,\alpha_j,\dots) + f(\dots,\alpha_j,\dots,\alpha_i,\dots)=0. \]

所以交换两列时，\(f\) 变号。

2. 某一列乘以常数
由列线性，若第 \(i\) 列乘以 \(c\)，则

\[ f(\dots,c\alpha_i,\dots)=c f(\dots,\alpha_i,\dots). \]

3. 某一列加上另一列的倍数
若第 \(i\) 列加上第 \(j\) 列的 \(c\) 倍，则

\[ f(\dots,\alpha_i+c\alpha_j,\dots,\alpha_j,\dots) = f(\dots,\alpha_i,\dots,\alpha_j,\dots) + c f(\dots,\alpha_j,\dots,\alpha_j,\dots). \]

第二项中第 \(i\) 列和第 \(j\) 列都是 \(\alpha_j\)，所以为 \(0\)。因此

\[ f(\dots,\alpha_i+c\alpha_j,\dots,\alpha_j,\dots) = f(\dots,\alpha_i,\dots,\alpha_j,\dots). \]

所以“倍加”不改变 \(f\) 的值。

因此，\(f\) 在初等列变换下的变化规律与行列式 \(\det\) 完全一致。

现在，任意矩阵 \(A\) 都可以通过一系列初等列变换化为对角矩阵

\[ D=\operatorname{diag}(d_1,d_2,\dots,d_n). \]

设这些列变换的总乘积因子为 \(k\)，则

\[ \det(D)=k\det(A),\qquad f(D)=kf(A). \]

另一方面，由列线性，

\[ f(D)=f(d_1e_1,d_2e_2,\dots,d_ne_n) = d_1d_2\cdots d_n f(I). \]

又因为归一性 \(f(I)=1\)，所以

\[ f(D)=d_1d_2\cdots d_n=\det(D). \]

于是

\[ kf(A)=f(D)=\det(D)=k\det(A). \]

由于 \(k\neq 0\)，得到

\[ f(A)=\det(A). \]

所以行列式函数 \(\det\) 与满足列线性、反对称性、归一性的函数 \(f\) 相同。

习题 2   三次方程的求根公式

第一小问

令

\[ F(x)=x^3+ax^2+bx+c. \]

显然 \(F(x)\) 在 \(\mathbb R\) 上连续。并且

\[ \lim_{x\to -\infty}F(x)=-\infty,\qquad \lim_{x\to +\infty}F(x)=+\infty. \]

所以存在 \(x_1

\[ F(x_1)<0

由介值定理，存在 \(\alpha\in(x_1,x_2)\)，使得

\[ F(\alpha)=0. \]

因此方程

\[ x^3+ax^2+bx+c=0 \]

总有实数根。

第二小问：化为缺二次项的三次方程

令

\[ x=y-\frac{a}{3}. \]

代入原方程，计算得

\[ x^3=\left(y-\frac a3\right)^3 = y^3-ay^2+\frac{a^2}{3}y-\frac{a^3}{27}, \] \[ ax^2=a\left(y^2-\frac{2a}{3}y+\frac{a^2}{9}\right) = ay^2-\frac{2a^2}{3}y+\frac{a^3}{9}, \] \[ bx=by-\frac{ab}{3}. \]

相加后，\(y^2\) 项消去，得到

\[ y^3+\left(b-\frac{a^2}{3}\right)y+ \left(\frac{2a^3}{27}-\frac{ab}{3}+c\right)=0. \]

所以

\[ \boxed{p=b-\frac{a^2}{3}},\qquad \boxed{q=\frac{2a^3}{27}-\frac{ab}{3}+c}. \]

方程化为

\[ y^3+py+q=0. \]

第三小问：增元降次

设

\[ y=u+v. \]

代入

\[ y^3+py+q=0, \]

得到

\[ u^3+v^3+(3uv+p)(u+v)+q=0. \]

取条件

\[ 3uv+p=0,\quad \text{即 } uv=-\frac p3. \]

此时

\[ u^3+v^3=-q. \]

令

\[ U=u^3,\qquad V=v^3. \]

则

\[ U+V=-q,\qquad UV=(uv)^3=\left(-\frac p3\right)^3=-\frac{p^3}{27}. \]

因此 \(U,V\) 是二次方程

\[ t^2+qt-\frac{p^3}{27}=0. \]

的根。

第四小问：解一个二次方程

解

\[ t^2+qt-\frac{p^3}{27}=0. \]

由求根公式，

\[ t=-\frac q2\pm\sqrt{\frac{q^2}{4}+\frac{p^3}{27}}. \]

记

\[ \Delta=\left(\frac q2\right)^2+\left(\frac p3\right)^3, \]

则

\[ u^3,\ v^3 = -\frac q2\pm\sqrt{\Delta}. \]

第五小问：得到 \(y\) 的解

设

\[ \alpha=\sqrt[3]{-\frac q2+\sqrt{\Delta}},\qquad \beta=\sqrt[3]{-\frac q2-\sqrt{\Delta}}, \]

并且选取立方根使得

\[ \alpha\beta=-\frac p3. \]

令

\[ \omega=e^{2\pi i/3}=-\frac12+\frac{\sqrt3}{2}i. \]

则

\[ \omega^3=1,\qquad 1+\omega+\omega^2=0. \]

可行的配对为

\[ (u,v)=(\alpha,\beta),\quad (u,v)=(\omega\alpha,\omega^2\beta),\quad (u,v)=(\omega^2\alpha,\omega\beta). \]

因此三个根为

\[ \boxed{ y_1=\alpha+\beta, } \] \[ \boxed{ y_2=\omega\alpha+\omega^2\beta, } \] \[ \boxed{ y_3=\omega^2\alpha+\omega\beta. } \]

第六小问：还原为原方程的根

由第二小问，

\[ x=y-\frac a3. \]

将第五小问中的 \(y_1,y_2,y_3\) 代入，并代入

\[ p=b-\frac{a^2}{3},\qquad q=\frac{2a^3}{27}-\frac{ab}{3}+c, \] \[ \Delta=\left(\frac q2\right)^2+\left(\frac p3\right)^3 = \left(\frac q2\right)^2+ \left(\frac{b-\frac{a^2}{3}}{3}\right)^3, \]

得到原方程的三个根：

\[ \boxed{ x_1=-\frac a3+ \sqrt[3]{-\frac q2+\sqrt{\Delta}}+ \sqrt[3]{-\frac q2-\sqrt{\Delta}} } \] \[ \boxed{ x_2=-\frac a3+ \omega\sqrt[3]{-\frac q2+\sqrt{\Delta}}+ \omega^2\sqrt[3]{-\frac q2-\sqrt{\Delta}} } \] \[ \boxed{ x_3=-\frac a3+ \omega^2\sqrt[3]{-\frac q2+\sqrt{\Delta}}+ \omega\sqrt[3]{-\frac q2-\sqrt{\Delta}} } \]

其中

\[ \boxed{ q=\frac{2a^3}{27}-\frac{ab}{3}+c, } \] \[ \boxed{ \Delta= \left(\frac q2\right)^2+ \left(\frac{b-\frac{a^2}{3}}{3}\right)^3, } \]

并且立方根的选取满足

\[ \boxed{ \sqrt[3]{-\frac q2+\sqrt{\Delta}} \cdot \sqrt[3]{-\frac q2-\sqrt{\Delta}} = -\frac{b-\frac{a^2}{3}}{3}. } \]

这就是三次方程

\[ x^3+ax^2+bx+c=0 \]

的卡尔达诺（Cardano）求根公式。
